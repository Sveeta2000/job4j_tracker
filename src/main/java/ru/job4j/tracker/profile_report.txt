Данный анализ был проведен с помощью инструмента jconsole.
Приложение StartUI запускалось с ограничениями памяти -Xmx7m -Xms7m, с целью более простого
воспроизведения OutOfMemoryError. При попытке добавления 1_000_000 заявок приложение вышло за пределы памяти.
Изменения состояния приложения отслеживались при помощи jconsole(см. profile_report.png)
